## Docker

### build - Nuxt ver.3.1.2

- [Nuxt3 × Docker 環境構築（正式版）](https://qiita.com/A-Kira/items/5ce3e1bff34e179ebbc2)

<!-- commandをコメントアウトする -->

$ docker-compose build

$ docker-compose up

$ docker-compose exec frontend bash

$ npx nuxi init . --force

$ npm install

$ npm run dev

$ exit

$ docker-compose down

<!-- commandのコメントアウトを外す -->

$ docker-compose up -d
